    3. **Basic use of pip:**
    - pip is a tool used to install and manage external Python packages (libraries) that are not included with Python by default.
      (ex. pandas, Numpy, flask, etc)
    - Basic use of pip = to install, update, and remove Python libraries.

4. **commands...**
    - Check pip version - *pip --version/ pip -v*
    - Install a package - *pip install package_name*
    - Uninstall a package - *pip uninstall package_name*
    - Upgrade an installed package - *pip install --upgrade package_name*
    - List installed packages - *pip list*
    - to check libraries version - *pip show <library_nm>*
