**Getting started with Numpy**

*step 1 :*

    Creating virtual environment :
 
        python -m venv .venv 


*step 2 :*

    Activating virtual environment :

        & .venv\Scripts\Activate.ps1    (here the & runs the script, ps1 is for powershell)

*step 3 :*

    installing Numpy library:

        pip install Numpy

    also Upgrade installed package (if needed) :
        pip install --upgrade numpy
